# Miller Inspector

Aplicación móvil para generar informes de inspección.

## Características
- Formulario para capturar datos de operario y proceso.
- Toma de fotografías con la cámara del dispositivo.
- Generación de informe en PDF con los datos y la foto tomada.
- Almacenamiento temporal del PDF y opción de compartirlo mediante aplicaciones disponibles.
- Tema oscuro personalizado.

## Compilación

Se requiere Flutter. Instale Flutter según la documentación oficial y ejecute:

```bash
flutter pub get
flutter run
```

La aplicación arranca directamente en la pantalla de "Nuevo Informe".
